# message

基于 Chalk 的 Message 提示。

## 使用说明

```bash
npm install -g @tlyboy/message
```

```javascript
import message from '@tlyboy/message'
```

```javascript
message.info('info')
message.success('success')
message.error('error')
message.warning('warning')
```
